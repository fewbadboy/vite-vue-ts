# Vue 3 + TypeScript + Vite

## Install

```shell
pnpm install --registry=https://registry.npmmirror.com

# mock server
pnpm run server

# run project
pnpm run dev

# update
# pnpm up axios@latest
pnpm up --latest
```

## Browser support

- **Chrome 111** (released March 2023)
- **Safari 16.4** (released March 2023)
- **Firefox 128** (released July 2024)
